Responsive CSS Cheat Sheet

Filters

filter: grayscale(100%);

Apply visual effects to images or elements.

Media Queries

@media (max-width: 768px) { ... }

Change styles based on viewport size.

Responsive Design

flexbox / grid / fluid units

Build layouts that adapt to different screens.

Typography

font-size: clamp(...)

Scale font sizes smoothly using clamp() and relative units.
